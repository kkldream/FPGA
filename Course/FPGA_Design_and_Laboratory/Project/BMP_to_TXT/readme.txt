bmp 8bit 2 arr ver 0.9 by kuiyou


Bmp8bit2arr.exe

1. 把libgg_s_dw2-1.dll複製到以下目錄
- for 32bit dll: c:\Windows\system32 or c:\Windows or the root directory of the affected application
- for 16bit dll: c:\Windows\system or c:\Windows or the root directory of the affected application

2. 用小畫家開圖，調整大小至適當像素（寬度取8的倍數）
3. 另存新檔為***.bmp，256色點陣圖（實際轉檔視為單色點陣圖）
4. 在檔案總管，直接將存檔後圖片拖曳至bmp8bit2arr.exe上
5. 若轉檔成功，會產生arr.txt文字檔，可直接使用於verilog程式中


可產生Verilog程式適用的圖檔


How To Use:
just drag image.
*BMP must be 8bit.
you can use win painter convert image to BMP 8bit.
*width must be 4's multiple
